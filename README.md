## Cara Install & Menjalankan webpack
Untuk menginstall beberapa depedencies & devdepedencies pada <b>package.json</b>
```
npm install
```
Untuk compile module bundler pada webpack
```
npm run build
```
Run pada mode development
```
npm run start-dev
```

Run pada mode production
```
npm run start-prod
```
